## 流畅度大幅优化 + 前置摄像头切换

### 流畅度优化(底层代码)
1. **H.264 编码器改用异步 Callback 模式** — 替代旧的 10ms 轮询循环,编码帧就绪后立即回调推送,端到端延迟降低约 10ms
2. **修复 MJPEG 多客户端广播 bug** — 之前多客户端共享单 Channel 导致抢帧丢失;现在每个客户端独立 Channel,互不影响
3. **MJPEG 帧头预计算** — boundary 字符串不再每帧分配,减少 GC 压力
4. **MJPEG 客户端反压策略优化** — 客户端缓冲区满时丢弃旧帧保留最新帧,保证实时性

### 新功能
1. **⚡ 流畅模式预设按钮** — 工具栏闪电图标,一键应用最优流媒体参数:
   - H.264 编码 + 720p 分辨率 + CBR 8Mbps + 30fps
2. **📷 浮动摄像头切换按钮** — 预览区域右上角圆形浮动按钮,点击循环切换前后摄像头,图标随当前朝向动态变化

### 推荐使用方式
1. 打开 app 进入摄像头页
2. 点击预览右上角圆形按钮切换到想要的摄像头(前/后)
3. 点击工具栏⚡闪电图标一键进入流畅模式
4. 用浏览器访问 `http://<手机IP>:8080/cam.h264` 观看 H.264 流

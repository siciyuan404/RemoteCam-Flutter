## 新功能

### 快捷工具栏
在预览区域下方新增快捷工具栏,包含三个常用控制:

- **📷 切换前后摄像头**:一键切换前置/后置摄像头(图标会根据当前朝向动态变化)
- **📐 比例选择**:支持 4:3 / 16:9 / 1:1 比例切换,自动选择该比例下最大分辨率
- **🎥 帧率选择**:24 / 30 / 60 fps 一键切换

### 技术改进
- `SensorDesc` 新增 `lensFacing` 字段(0=后置, 1=前置, 2=外接),用于精确判断摄像头朝向
- 比例切换基于当前摄像头支持的分辨率列表自动计算可用比例
- 帧率切换通过 `AppState.updateTargetFps()` 统一管理状态

## 兼容性
- Android 10.0+ (API 29)
- arm64-v8a 架构
